Multilevel Image Segmentation Using Modified Red Deer Algorithm

Meta-Herustics
Evolutionary Algorithms
Particle Swarm Optimization
Red Deer Algorithm
Multilevel Image Thresholding


This paper introduces a Modified Red Deer Algorithm (MRDA) using an evolution strategy to optimize segmentation for grayscale images. Inspired by the behavior of red deer during their breeding season, the original Red Deer Algorithm (RDA) is a well-known meta-heuristic for solving complex optimization problems.
