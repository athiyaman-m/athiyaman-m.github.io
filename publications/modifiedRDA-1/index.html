A New Modified Red Deer Algorithm for Multi-level Image Thresholding

Meta-Herustics
Evolutionary Algorithms
Genetic Algorithm
Red Deer Algorithm
Multilevel Image Thresholding


This paper introduces a Modified Red Deer Algorithm (MRDA) using an evolution strategy to optimize threshold selection for grayscale images. Inspired by the behavior of red deer during their breeding season, the original Red Deer Algorithm (RDA) is a well-known meta-heuristic for solving complex optimization problems.

Author
